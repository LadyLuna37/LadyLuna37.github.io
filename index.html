HTML LIST REFERENCE


List Tags

Tag Name Description
<ul> Unordered List Defines an Unordered List
<ol> Ordered List Defines an Ordered List
<li> List Item Defines a List Item
<dl> Description List Defines a Description List
<dt> Description List Term Defines a term in a Description List
<dd> Description List Term Description Describes the term in a Description List


Ordered List "Type" Attributes

Attribute Name Description
type="1" Numbered List The list items will be numbered with numbers (default)
type="A" Uppercase Numbering The list itmes will be numbered with uppercase letters
type="a" Lowercase Numbering The list items will be numbered with lowercase letters
type="I" Uppercase Roman Numerals The list items will be numbered with uppercase roman numerals
type="i" Lowercase Roman Numerals The list items will be numbered with lowercase roman numerals
